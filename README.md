# Explore Togther
    An social media app for traveller

## Feature
>User Feed
* List all user posts
* Sort by Date & Trending

>Create a Post

>Like a post

>Edit & Delete Post

>User Profile

* Add Avatar
* Add Bio
* Add Portfolio URL
* Follow/Unfollow

>Explore Feed
* Bookmark Post
>Authentication
* Sign-up
* Login
* Logout

## Technology Used
> React JS
* React Context API
* React useReducer
* React Router

> CSS

> HTML5
